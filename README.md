# Project Overview: What is This Logistics Platform?
A B2B Logistics Management Platform that helps businesses track shipments, manage warehouses, and handle logistics operations. Think of it as a simplified version of DHL/FedEx/Grab Express management system.
### Key Features to Showcase:

Real-time shipment tracking
Warehouse management
Delivery management
Analytics dashboard
User profiles/accounts
